**PC Death** is a modern web application designed to help users organize and track everything they need to reinstall after formatting their computer. With a cyberpunk aesthetic and intuitive interface, it transforms the tedious task of system reinstallation into a streamlined, manageable process.

Whether you're doing a clean OS install, setting up a new machine, or recovering from a system failure, PC Death ensures you never forget critical programs, drivers, extensions, or configurations.

## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher) - [Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- npm or bun package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MarcosAlves90/pc-death.git
```

2. Navigate to the project directory:
```bash
cd pc-death
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal)

## ğŸ“– Usage

1. **Select Protocol**: Choose "RED DEATH" to start a format & reinstall protocol
2. **Add Groups**: Create custom groups to organize your items (Programs, Drivers, Extensions, etc.)
3. **Add Items**: Within each group, add items you need to reinstall with:
   - Item name
   - Priority level (High, Medium, Low)
   - Optional notes
4. **Track Progress**: As you reinstall each item, mark it as completed or skipped
5. **Export/Import**: Backup your lists by exporting to JSON, or import previous backups
